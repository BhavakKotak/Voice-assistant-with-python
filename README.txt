1.PYTTSX3  (pip install pyttsx3) from cmd - convert voice to text and vice versa
2.datetime (inbuilt)- get datetime of system
3.speech_recognition (pip install speechrecognition) from cmd
4.wikipedia (pip install wikipedia) 
5. webbrowser(inbuilt)
6.os (inbuilt) 	
7.selenium (pip install selenium) - for automation
8. time (inbuilt)
